SQL Server Interview Preparation

Basic SQL Server Concepts

  • DDL: Data Definition Language (CREATE, ALTER, DROP)
  • DML: Data Manipulation Language (SELECT, INSERT, UPDATE, DELETE)
  • DCL: Data Control Language (GRANT, REVOKE)
  • TCL: Transaction Control Language (COMMIT, ROLLBACK, SAVEPOINT)

Keys in SQL

  • Primary Key: Uniquely identifies each record, not null + unique
  • Foreign Key: References primary key in another table
  • Unique Key: Ensures all values are unique but allows one null
  • Difference: Primary = Not Null + Unique; Unique = Unique only; Foreign = Reference

Delete vs Truncate vs Drop

  • DELETE: DML, can use WHERE, can rollback
  • TRUNCATE: DDL, removes all rows, can't use WHERE
  • DROP: DDL, removes table/schema entirely

Constraints

NOT NULL, UNIQUE, PRIMARY KEY, FOREIGN KEY, CHECK, DEFAULT

Aggregate Functions

SUM, COUNT, AVG, MAX, MIN

Group By & Having

GROUP BY: Used with aggregate functions to group result-set

HAVING: Filters records after grouping (like WHERE but for groups)

Joins

INNER, LEFT, RIGHT, FULL OUTER, CROSS JOIN

Stored Procedure vs Function

  • SP: Can perform operations, return multiple values, support transactions
  • Function: Must return a value, cannot modify data (except inline TVFs)

Types of Functions

  • Scalar Function
  • Inline Table-valued Function
  • Multi-Statement Table-valued Function

Views

Virtual table created using SELECT queries. Used for abstraction and simplification.

Triggers

Automatic execution of actions on INSERT, UPDATE, DELETE events

Cursors

Row-by-row processing of query results (avoid if possible due to performance)

CTE (Common Table Expression)

Temporary result set used within SELECT, INSERT, UPDATE, DELETE

Normalization

Process of organizing data to reduce redundancy. Normal forms: 1NF, 2NF, 3NF, BCNF

Subquery vs Join

  • Subquery: Query within another query (inner query runs first)
  • Join: Combines data from multiple tables based on condition

Transactions and Rollback

Used to group SQL operations into a single unit. Can COMMIT or ROLLBACK
